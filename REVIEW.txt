D) RUN & TEST GUIDE (WINDOWS)
1. Prerequisites
   - Install JDK 17 and ensure `java -version` reports 17.x.
   - Install Apache Maven (`mvn -version`).
   - Install MySQL Server 8.x and create a user with privileges for the nova_exam schema.
   - Install Apache Tomcat 10.x (Jakarta EE 9 compatible), e.g., `C:\apache-tomcat-10.x`.
2. Database setup
   - Edit `src/main/java/com/nova/dao/DBConnection.java` with your MySQL username and password.
   - Run schema script: `mysql -u root -p < db_init.sql`.
   - Verify tables: `mysql -u root -p nova_exam -e "SHOW TABLES;"`.
3. Build the application
   - `cd D:\NovaOnlineExamSystem`
   - `mvn clean package`
   - (Completed successfully in this review with `mvn -q -DskipTests clean package`).
4. Deploy to Tomcat
   - Copy `target\NovaOnlineExamSystem.war` to `C:\apache-tomcat-10.x\webapps\`.
5. Start/stop Tomcat
   - Start: `C:\apache-tomcat-10.x\bin\startup.bat`
   - Stop: `C:\apache-tomcat-10.x\bin\shutdown.bat`
6. Test in browser (after deployment)
   - Landing: http://localhost:8080/NovaOnlineExamSystem/index.jsp
   - Student register: http://localhost:8080/NovaOnlineExamSystem/student_register.html
   - Student login: http://localhost:8080/NovaOnlineExamSystem/student_login.html
   - Teacher login: http://localhost:8080/NovaOnlineExamSystem/teacher_login.html
   - Start exam (student) : http://localhost:8080/NovaOnlineExamSystem/start-exam
   - Teacher dashboard: http://localhost:8080/NovaOnlineExamSystem/view-questions
7. Postman samples (x-www-form-urlencoded body)
   - Student login: POST http://localhost:8080/NovaOnlineExamSystem/login (fields: email, password)
   - Teacher login: POST http://localhost:8080/NovaOnlineExamSystem/teacher-login (fields: email, password)
   - Add question: POST http://localhost:8080/NovaOnlineExamSystem/add-question (fields: questionText, optionA, optionB, optionC, optionD, correctOption) â€“ requires teacher session cookie.
8. MySQL verification queries
   - `SELECT id, name, email FROM student;`
   - `SELECT id, question_text, correct_option FROM question ORDER BY id DESC;`
   - `SELECT student_id, score FROM result ORDER BY id DESC;`
   - `SELECT id, email FROM teacher;`
9. Logs & troubleshooting
   - Check `C:\apache-tomcat-10.x\logs\catalina.out` (or `logs\localhost.<date>.log`) for servlet/runtime issues.
   - Tomcat manager (optional) helps confirm deployment status.
